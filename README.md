# Random Note Picker

## Description
A random note picker that displays quotes, facts, and interesting notes randomly from a collection.

## Features
- Randomly picks notes from data.txt
- Displays quotes, facts, and interesting content
- Simple and easy to use

## Files
- `main.py` - Main application entry point
- `utils.py` - Helper functions for random selection
- `data.txt` - Database of quotes, facts, and notes

## Requirements
- Python 3.x
- No external libraries needed

## Usage
```bash
python main.py
```

## Setup

Make sure data.txt exists in the same directory with one note per line.
